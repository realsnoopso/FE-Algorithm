# 1. 시저 암호

- 각 알파벳을 일정한 거리만큼 밀어서 다른 알파벳으로 바꾸는 방식
- s: 원래 글자 (소문자, 대문자, 공백)
- n: 이동 거리
- result: n만큼 이동
- AB (1) -> BC
- AB (3) -> DE
- z (1) -> a
- aBz (4) -> eFd
- 공백은 아무리 밀어도 공백

- array로 변경 후 reduce
- type을 확인
  - if 공백
  - 공백 그대로 반환
  - if 대문자 or 소문자
  - 숫자로 변환 후 n만큼 더하기
  - 다시 문자로 변환
- string으로 합치기
